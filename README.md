# GraphQL Apollo Server Template

## How to use this template

1. Clone repository
2. Run `npm i`
3. Rename `.env.temp` to `.env`
4. Replace values in `.env`
5. Delete `.git` folder
6. Run `git init`
7. Run `git add .`
8. Run `git commit -m "initial commit"`

## Including
